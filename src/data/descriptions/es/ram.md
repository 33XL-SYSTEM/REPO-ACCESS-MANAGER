# R.A.M - Repo Access Manager

Repo Access Manager (R.A.M) es el centro principal y orquestador de todos mis repositorios. El objetivo principal de este sistema es servir como un catálogo visual de alto rendimiento, evitando la necesidad de buscar manualmente entre docenas de repositorios en GitHub o GitLab.

Construido con React, TypeScript y Tailwind, R.A.M presenta una estética oscura, moderna y altamente tecnológica (brand-black), organizando proyectos por categoría, estado y tecnología.

## 🛠️ Características Principales

- **Catálogo de Proyectos:** Visualización centralizada de proyectos Web, Móvil, Escritorio y Juegos.
- **Acceso Rápido:** Enlaces directos a repositorios públicos, vistas previas en vivo y documentación detallada.
- **Seguimiento de Estado:** Indicación visual del estado del proyecto (activo, en progreso, archivado).
- **Soporte Multilingüe (i18n):** Soporte nativo para Portugués, Inglés y Español.
- **Detalles Dinámicos:** Modales y entornos integrados para leer las especificaciones técnicas del proyecto sin salir del hub.
