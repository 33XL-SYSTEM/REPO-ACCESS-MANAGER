# AÇAÍ TOP LANCHES // Plataforma Integrada de Ventas y E-commerce

**Açaí Top Lanches** es una robusta plataforma de gestión operativa y e-commerce hecha a medida para el sector de la alimentación y el delivery. El sistema nació para modernizar el proceso de compra y optimizar el flujo de trabajo de una cafetería real, eliminando la dependencia de los intermediarios de entrega tradicionales.

La arquitectura de software altamente escalable cuenta con un front-end moderno en **React 19, TypeScript y Vite**, y un ecosistema backend híbrido utilizando rutas API sin servidor alojadas en Vercel e integradas con Vercel Postgres.

La UX incluye un Sistema Constructor de Vasos para personalización en tiempo real y un Pago Híbrido que dispara mensajes estructurados a través de la API de WhatsApp. El panel administrativo presenta gráficos y chat P2P a través de PeerJS.
