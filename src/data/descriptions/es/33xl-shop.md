# 33XL SHOP - E-commerce de Afiliados

33XL SHOP es un escaparate inteligente y ultra estilizado desarrollado para la curación de enlaces de afiliados. El proyecto se centra en la escalabilidad, el diseño inmersivo y la conversión, lo que permite mostrar miles de productos en docenas de nichos sin saturar la interfaz de usuario.

## 📖 Acerca del Proyecto

Muchos afiliados luchan por organizar sus enlaces. El objetivo principal de 33XL SHOP no es ser un comercio electrónico tradicional con un carrito de compras, sino un centro de curación premium. Cuando un visitante se interesa en un producto, hace clic en "Ver Oferta" y es redirigido instantáneamente a la tienda asociada.

## 🎨 Estética e Identidad Visual

La interfaz presenta elementos del Steampunk Victoriano mezclados con diseño web moderno (Glassmorphism, bordes translúcidos, fondos borrosos y micro-interacciones).

## 🚀 Tecnologías Utilizadas

- **React 19** - Biblioteca de interfaz de usuario súper rápida.
- **TypeScript** - Tipado estático.
- **Vite 8** - Empaquetador de alto rendimiento.
- **Tailwind CSS v4** - Framework de utilidades.
- **Lucide React** - Iconos SVG.

## 🌟 Hero Dinámico

El componente `<Hero />` reacciona activamente a la categoría seleccionada por el usuario, cambiando instantáneamente el título, el subtítulo y la imagen de fondo temática.
