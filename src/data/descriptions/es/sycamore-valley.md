# SYCAMORE VALLEY GAME // Terror y Exploración Paranormal

**Sycamore Valley** es un juego de terror y exploración en 2D desarrollado bajo una densa estética retro de pixel-art. El jugador asume el papel del Agente Especial Evan Brecht de la Agencia de Investigaciones Paranormales (AIP), encargado de investigar la misteriosa desaparición de civiles y enfrentarse a una entidad hostil conocida como Yaldabaoth.

La jugabilidad combina el encanto clásico de los juegos de aventura con vista superior con mecánicas modernas de combate dinámico, gestionadas por un Sistema de Resistencia y Enfoque. Además del combate físico, se basa en gran medida en acertijos de investigación y en la recopilación de pistas documentales.

Técnicamente, el motor gráfico opera de forma nativa y optimizada en la web, utilizando la Web Audio API para efectos de audio espacializados que construyen la tensión sonora.
