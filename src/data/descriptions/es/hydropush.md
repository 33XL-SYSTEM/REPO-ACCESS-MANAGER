# APLICACIÓN HYDROPUSH

Aplicación de monitorización y fomento de la hidratación desarrollada para recordar a los usuarios que beban agua regularmente.

Construida con React Native, Capacitor y Firebase, esta aplicación ofrece soporte multiplataforma con un único código base. Cuenta con notificaciones push, seguimiento de objetivos diarios y elementos de gamificación para mantener a los usuarios involucrados y adecuadamente hidratados.
