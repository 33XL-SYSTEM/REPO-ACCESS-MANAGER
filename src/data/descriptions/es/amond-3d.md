# Motor Amond 3D

Un visor de modelos 3D altamente optimizado y plataforma interactiva construida con Three.js y WebGL. Maneja geometrías complejas, iluminación avanzada (PBR) y efectos de postprocesamiento directamente en el navegador.

Este motor permite cargar modelos GLTF/GLB personalizados, manipularlos en tiempo real y aplicar diferentes mapas de entorno (HDRI) y materiales. Está diseñado como una pieza fundamental para futuras aplicaciones web 3D y juegos.
