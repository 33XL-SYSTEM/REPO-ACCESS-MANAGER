# BANCO SIC MUNDUS

Un simulador bancario retro-futurista e inmersivo construido en C compilado a WebAssembly (WASM) y envuelto en React 19. Inspirado en la estética narrativa de la serie "Dark".

La aplicación simula un terminal bancario de una línea de tiempo alternativa, proporcionando una interfaz misteriosa y altamente técnica. Cuenta con transacciones seguras simuladas, una intrincada máquina de estados escrita en C para un alto rendimiento y un diseño UI/UX atmosférico con efectos de monitor CRT y sonidos analógicos.
