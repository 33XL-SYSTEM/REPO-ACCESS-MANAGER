# REI DAS MORDIDAS // Portal Gastronómico de Alto Rendimiento

**Rei das Mordidas** es una plataforma y blog de recetas gastronómicas premium diseñado para ofrecer una experiencia digital inmersiva y de alto rendimiento. La identidad visual de la interfaz se basa en la Estética Rica y los principios del diseño moderno.

Cuenta con un efecto dinámico patentado en CSS llamado **Extreme Jelly Movement**, creando una animación orgánica y fluida en el fondo. Funcionalmente, tiene un Sistema de Compartir Inteligente utilizando la Web Share API nativa.

La gestión de contenido es limpia y modular a través de Markdown con Frontmatter, procesado y convertido en la interfaz gráfica a través de tuberías de renderizado estático, asegurando tiempos de respuesta ultrarrápidos y excelente SEO.
