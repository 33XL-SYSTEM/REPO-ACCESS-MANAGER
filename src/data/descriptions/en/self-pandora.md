# Pandora's Self // Psychological 3D Platformer

**Pandora's Self** is an abstract and psychological 3D platform game focused on the integration of the inner Shadow. The gameplay revolves around philosophical concepts, where the player explores dimensional biomes and interacts with philosophical NPCs scattered across the multiverse.

Technically, the project merges the WebGL 3D rendering of Three.js with the interface, logic, and dialogs of Phaser 3, operating transparently in an unusual hybrid architecture. The graphical interface of Zebrão_OS was developed over the DOM using HTML/CSS and Tailwind, creating complex cyberpunk menus without overloading the main rendering engine.
