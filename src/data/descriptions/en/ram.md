# R.A.M - Repo Access Manager

Repo Access Manager (R.A.M) is the central hub and orchestrator of all my repositories. The main objective of this system is to serve as a high-performance visual catalog, avoiding the need to search through dozens of repositories manually on GitHub or GitLab.

Built with React, TypeScript, and Tailwind, R.A.M features a dark, modern, and highly technological aesthetic (brand-black), organizing projects by category, status, and technology.

## 🛠️ Main Features

- **Project Catalog:** Centralized visualization of Web, Mobile, Desktop, and Game projects.
- **Quick Access:** Direct links to public repositories, live previews, and detailed documentation.
- **Status Tracking:** Visual indication of project status (active, in-progress, archived).
- **Multilingual Support (i18n):** Native support for Portuguese, English, and Spanish.
- **Dynamic Details:** Modals and embedded environments to read project technical specifications without leaving the hub.
