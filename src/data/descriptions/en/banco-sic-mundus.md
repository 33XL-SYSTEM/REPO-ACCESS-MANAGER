# BANCO SIC MUNDUS

A retro-futuristic and immersive banking simulator built in C compiled to WebAssembly (WASM) and wrapped in React 19. Inspired by the narrative aesthetics of the "Dark" series.

The application simulates a banking terminal from an alternate timeline, providing a mysterious and highly technical interface. It features secure simulated transactions, an intricate state machine written in C for high performance, and an atmospheric UI/UX design with CRT monitor effects and analog sounds.
