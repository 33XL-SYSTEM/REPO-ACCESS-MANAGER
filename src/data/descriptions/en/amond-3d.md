# Amond 3D Engine

A highly optimized 3D model viewer and interactive platform built with Three.js and WebGL. It handles complex geometries, advanced lighting (PBR), and post-processing effects directly in the browser.

This engine allows loading custom GLTF/GLB models, manipulating them in real-time, and applying different environmental maps (HDRI) and materials. It's designed as a foundational piece for future 3D web applications and games.
