# 33XL SHOP - Affiliate E-commerce

33XL SHOP is a smart, ultra-stylized showcase developed for affiliate links curation. The project is focused on scalability, immersive design, and conversion, allowing the administrator to display thousands of products divided into dozens of niches without cluttering the UI.

## 📖 About the Project

Many affiliates struggle to organize their links. The main goal of 33XL SHOP is not to be a traditional e-commerce with a shopping cart, but rather a premium curation hub. When a visitor is interested in a product, they click "View Offer" and are instantly redirected to the partner store.

## 🎨 Aesthetics and Visual Identity

The interface features Victorian Steampunk elements mixed with modern web design (Glassmorphism, translucent borders, blur backgrounds, and micro-interactions).

## 🚀 Technologies Used

- **React 19** - Super fast UI library.
- **TypeScript** - Static typing.
- **Vite 8** - High-performance bundler.
- **Tailwind CSS v4** - Utility framework.
- **Lucide React** - SVG icons.

## 🌟 Dynamic Hero

The `<Hero />` component actively reacts to the user's selected category, instantly changing the title, subtitle, and thematic background image.
