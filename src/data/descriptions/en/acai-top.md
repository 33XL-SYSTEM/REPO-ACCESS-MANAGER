# AÇAÍ TOP LANCHES // Integrated E-commerce and Sales Platform

**Açaí Top Lanches** is a robust e-commerce and operational management platform tailor-made for the food and delivery sector. The system was born to modernize the purchasing journey and optimize the workflow of a real snack bar, eliminating the dependence on traditional delivery intermediaries.

The highly scalable software architecture features a modern front-end in **React 19, TypeScript, and Vite**, and a hybrid backend ecosystem using Serverless API routes hosted on Vercel integrated with Vercel Postgres.

The UX includes a Cup Builder System for real-time customization and a Hybrid Checkout that fires structured messages via WhatsApp API. The administrative dashboard features charts and P2P chat via PeerJS.
